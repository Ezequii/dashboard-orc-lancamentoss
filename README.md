# Dashboard de Orçamentos — v4.6 TV

Revisão específica para visualização em TV de 50 polegadas, monitores Full HD e telas 4K.

## Melhorias para TV
- Tipografia ampliada em telas com largura a partir de 1600 px.
- KPIs, filtros, tabela, fluxo, legendas e rótulos dos gráficos maiores.
- Sidebar e ícones ampliados.
- Gráficos de barras mais altos e com nomes maiores.
- Botão **Ocultar filtros** para aproveitar melhor a altura da TV.
- Filtros ficam recolhidos por padrão em telas grandes e podem ser reabertos.
- Botão de tela cheia no cabeçalho.
- Ajuste adicional para telas 4K.
