# Dashboard de Orçamentos — v4.7 TV Completa

Versão preparada para mostrar **todos os gráficos simultaneamente em uma TV Full HD de 50 polegadas**, quando os filtros estiverem recolhidos.

## Organização na TV
- Cabeçalho compacto.
- Filtros recolhidos por padrão.
- Cinco indicadores em uma linha compacta.
- Primeira linha: serviços por líder e valor por fornecedor.
- Segunda linha: fluxo dos processos e valor por etapa.
- Terceira linha: recebidos x lançados.
- Sem rolagem vertical no modo TV em tela cheia.
- Tipografia continua ampliada para leitura à distância.

## Uso recomendado na TV
1. Abra o dashboard.
2. Use o botão de tela cheia.
3. Mantenha os filtros ocultos para visualizar todo o painel.
4. Abra os filtros somente quando precisar alterar a seleção.
